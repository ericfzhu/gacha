# Retro Arcade Visual Direction

## Goal
Shift UI presentation from clean productivity style to a coherent retro-arcade identity.

## Core look
- Deep-space gradient backgrounds with scanline/grid overlays.
- Neon palette accents (cyan, magenta, amber) on dark panels.
- Pixel/techno typography for headers and menu actions.
- High-contrast outlined cards and CRT-like glow pulses.

## Interaction style
- Large arcade menu buttons with hover glow and clear state shifts.
- Toast-style status messages for system feedback.
- Minimal text density on primary screens; prioritise mode selection and action speed.

## Implemented now
- New `HomeScene` as first entry after boot with retro arcade styling.
- Login, settings, and main navigation options surfaced on home screen.

## Next visual passes
- Re-skin `TitleScene`, `BattleScene`, and `FleetScene` with shared retro tokens.
- Replace Notion-like whites with dark neon panel system.
- Add shader-like effects for transitions and battle phase banners.
